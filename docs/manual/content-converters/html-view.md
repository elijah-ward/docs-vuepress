% HTML View Converter
% Greg Schueler

Allows HTML embedded in log data to be rendered.

Note: this plugin performs no changes, but passes HTML to be sanitized and rendered.

See: [[page:manual/content-converters/index.md#sanitized-html-output]].

## Input Data Types

* `text/html` - `String`

## Output Data Types

* `text/html` - `String`
